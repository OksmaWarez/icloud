# Using 32bit-Bruteforce-Passcode

## Overview

Using 32bit-bruteforce-passcode is a universal way of bruteforcing 4-6 digit passcodes on 32-bit iOS devices. It is commonly used for easier entering into pwnDFU (kDFU) mode and app preservation by jailbreaking.

## Prerequisites

### What You'll Need

A computer running Linux or macOS

A copy of [32bit-Bruteforce-Passcode](https://github.com/tuanemss/32bit-Bruteforce-Passcode/releases/tag/new/)

## Step-by-Step Guide

### Step 1: Setting Up

Download the utility from the link provided, and extract it to an easily accessible folder on your computer.

Put the device into DFU mode, then follow step 2.

### Step 2: Using the tool

Open up terminal, and navigate to where you extracted the .zip file.

Then, run `./bruteforce.sh` in terminal.

From there, the script should continue the bruteforcing automatically. Your passcode should be displayed on screen after some time.

### Step 3: Reboot

Reboot your device by either holding the power and home buttons for 10 seconds. Your device should now boot into the lock screen, where you can enter your bruteforced passcode.
